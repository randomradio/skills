# Setup Match — Gate 2  (atomic)

**Purpose:** A trustworthy edge requires a repeatable pattern with enough samples to validate.
Specialize in 1–2 setups; breadth fragments the sample and hides whether anything has an edge.

**Procedure**
1. Require the user's pre-defined setup list (e.g. gap-and-go, bull-flag breakout, short-into-spike,
   VWAP reclaim, support reclaim). If none is defined, that is the first task — define and backtest it.
2. Is the chart a **clean** instance of one of them?

**Verdict:** `GO` = MATCH → proceed · `NO-GO` = NO-MATCH ("interesting, but not my setup") → pass.
**Next:** `trade-planner`.
---

*Scope: process only — not financial advice, no buy/sell calls, no price predictions. Trading is treated as a logical, rule-based system; an urge to deviate is a signal to defer to the plan, never an input. Correct process is necessary, not sufficient: most day traders lose net of costs.*
